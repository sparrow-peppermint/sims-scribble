# Sims Scribble 98
## A digital drawing and guessing game based on the party game Gartic Phone

### To run the program
#### **From the command line**

```
npm install
npm run knex migrate:latest
npm run dev
```

You can find the server running on [http://localhost:3000](http://localhost:3000).
